# Madera Design — мобильное приложение

Мобильное приложение компании **Madera Design** для заказа корпусной мебели на заказ в г. Душанбе.

## Технологии

- React Native
- Expo
- React Navigation

## Структура экранов

- Стартовый экран (StartScreen) — приветствие, логотип, фото модели
- Экран входа (LoginScreen) — номер телефона, пароль
- Регистрация (RegisterScreen)
- Главная (HomeScreen)
- Каталог (CatalogScreen)
- Оформить заказ (OrderScreen)
- Личный кабинет (ProfileScreen)
- Ещё (MoreScreen) — разделы: О компании, Партнёрам, Акции, Контакты и др.
- Нижнее меню (MainTabs) — вкладки: Главная, Каталог, Заказ, Профиль, Ещё
